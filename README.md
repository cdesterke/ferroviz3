# ferroviz3
R package to perform ferroptosis analyses on limma output cross with mitochondria and metabolism databases


## code to install rcd package
```r
library(devtools)
install_github("cdesterke/ferroviz3")
```


## load data
```r
library(ferroviz3)
load(resulths)
load(ferrdb)

```
